# Бобруйский уезд (ок. 1800) — карта землевладений

Интерактивная карта Генерального плана Бобруйского уезда, где каждый участок
раскрашен по владельцу. Поиск по владельцам, гербы и церковные кресты, оригиналы
записей из алфавита владельцев, внешние ссылки.

**Открыть карту:** см. GitHub Pages этого репозитория.

## Управление
- **Поиск** — найти владельца; клик по нему центрирует его участок(и).
- **заливка / номера / гербы** — слои карты.
- Клик по участку — окно с владельцем(ами). У совместных участков владельцы
  перечислены равными долями; клик по имени раскрывает запись (оригинал, герб, ссылки).

## Источник
РГИА, ф. 1350, оп. 312, д. 89 (Минская губ.), л. 123–125 — «Алфавитъ всемъ
владельцамъ Бобруйскаго повета»; Генеральный план Бобруйского уезда.

## Технически
Статический сайт: [Leaflet](https://leafletjs.com/) (`CRS.Simple`); скан разбит на
тайлы (zoom 0–6). Данные — `data/owners.json` (владельцы) и `data/parcels.geojson`
(полигоны участков; координаты — пиксели скана). Раскладка по номеру участка.
